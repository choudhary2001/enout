Package configuration for Enout mobile app (Expo/React Native).
Package name: mobile
Version: 1.0.0

SCRIPTS
-------
- start: expo start - Start Expo development server
- android: expo start --android - Start Expo with Android target
- ios: expo start --ios - Start Expo with iOS target
- web: expo start --web - Start Expo with web target
- build:android: eas build --platform android --profile preview - Build preview APK via EAS
- build:android:prod: eas build --platform android --profile production - Build production APK via EAS
- build:apk: eas build --platform android --profile preview --local - Build APK locally
