HOW TO RUN ON ANDROID EMULATOR
-------------------------------

Method 1: Expo Go (Recommended - No Build Required)
1. Install Expo Go on emulator:
   - Open Google Play Store in emulator
   - Search "Expo Go"
   - Install the app
   
2. Start Expo server:
   cd /Users/divakarchoudhary/Downloads/enout-main/mobile
   npx expo start --android --clear

3. In Expo Go app:
   - Scan QR code from terminal
   - Or enter expo:// URL manually
   - App loads directly without build

Method 2: Native Build (Requires Disk Space)
1. Ensure minimum 10GB free disk space
2. Install Android SDK Build-Tools 34 (already installed)
3. Configure for SDK 34 instead of 36
4. Run: npx expo prebuild --platform android --clean
5. Run: npx expo run:android
